FAQ

Robinhood Chain · pigs · $OINK · claims

What are PixelPigs?

3,333 pixel-art pigs on Robinhood Chain. Pure code — no AI. Each pig is an ERC-721 NFT you can hold, trade on OpenSea, and use to claim $OINK.

Mint at hoodpixelpigs.xyz · max 11 pigs per wallet · price targets ~$1 USD in ETH (set on-chain).

How do I add Robinhood Chain to MetaMask?

  • Open MetaMask → network dropdown → Add network
  • Or tap ADD ROBINHOOD CHAIN on the mint page

Use these settings:

  • Network name: Robinhood Chain
  • RPC URL: https://rpc.mainnet.chain.robinhood.com
  • Chain ID: 4663
  • Currency: ETH
  • Explorer: https://robinhoodchain.blockscout.com

You need a small amount of ETH on Robinhood Chain for mint gas + mint price. Bridge ETH to chain 4663 before minting.

Which wallet can I use?

Any EVM wallet on chain 4663: MetaMask, OKX, Rabby, Robinhood Wallet, etc. On mint and claim pages, tap CONNECT and pick your wallet from the list.

What is $OINK?

$OINK is the PixelPigs community token on Robinhood Chain. Holders claim it once per pig they own. Fixed supply: 335,700,000 OINK.

Token: 0xF2c5…745b · Claim: oink.html

How much OINK do I get per pig?

Normal pig → 100,000 OINK
Legendary pig → 300,000 OINK (3×)
12 legendaries in the collection — see legendary board

Max if all 3,333 mint + claim: 3,321 × 100k + 12 × 300k = 335,700,000 OINK (entire supply).

Can someone game the claim with multiple wallets?

Not on the same pig. Claims are tied to token ID, not “forever owner.”

  • Each pig ID can be claimed only once — ever
  • Mint → claim → sell pig → buyer cannot claim again for that ID
  • Sending one pig through many wallets does not mint extra OINK

Using many wallets to mint more pigs (11 each) is separate — each pig still gets only one claim. That is a mint-cap question, not a claim exploit.

I bought a pig on OpenSea. Can I claim OINK?

Only if that token ID was never claimed before. Check claim page after connect — if it shows 0 claimable, the previous owner already claimed. You own the art; the one-time OINK for that ID is gone.

Are pigs transferable?

Yes. PixelPigs are full ERC-721 NFTs — trade, send, list on OpenSea anytime. Transfers do not reset or duplicate OINK claims.

Why does mint price show more or less than $1?

Mint price is set in ETH on-chain. The site converts live ETH/USD so it targets ~$1. When ETH moves, the ETH amount stays until the team updates setMintPrice — refresh the mint page to see current USD.
